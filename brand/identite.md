# Malentendu — identité de marque

## Architecture
| Élément | Rôle |
|---|---|
| **groove-engineer** | le **moteur** — produit open source (AGPLv3), l'infra qu'on pitche |
| **Malentendu Records** | le **label** — l'arm artistique, la preuve, existe déjà |
| **Le Malentendu** | la **musique produite** (le « genre ») — « c'est du Malentendu » |
| **« non = malentendu »** | la **signature / devise** |

## Sous-titre
> groove-engineer — *le moteur de Malentendu Records*

## Signature
> **non = malentendu**

*(Homophone volontaire non / nom : le nom EST un malentendu.)*

## Notes d'usage
- **Le nom de la musique** : « Le Malentendu » (singulier, avec article). Pour les morceaux : « des malentendus ».
- **La devise est de l'art, pas une posture de vente.** Sur le mur / dans l'œuvre : *non = malentendu* — chaque refus devient matière. Dans la pièce, face à un acheteur : un non reste un non, on écoute.
- Ton : situationniste, détournement, l'élégance du faux assumé.
- « Muzak » écarté (marque déposée vivante, Mood Media) — l'esprit gardé, le mot changé.
