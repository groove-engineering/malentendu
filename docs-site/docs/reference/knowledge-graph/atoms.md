# Atoms — the genre encyclopedia

An **atom** is a genre. It carries musicological facts (sourced, falsifiable), constraints (conventions held by the maker), felt responses (the circle's experience), and a political reading (owned by the author).

Fixing an atom fixes **all** its crossings. The curation lever is the atom (~600 genres), not the molecule (360,000 fusions).

## Status legend

| Status | Meaning |
|--------|---------|
| `proposed` | Drafted, not yet vetted by a human musicologist |
| `validated` | A human signed off |
| `contested` | Divergence held in storage — to resolve |
