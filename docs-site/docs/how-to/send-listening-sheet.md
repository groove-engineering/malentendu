# Listening sheet — template

> Copy and adapt for each track. Remove the `[brackets]`.

---

🎵 **[Title]** — [Genre A] × [Genre B]

**Listen:** [SoundCloud link]

**Leave feedback:** [open the Discussion on GitHub]

---

What I'm looking for from you:

**🎼 The ear** — Do you recognise [Genre A]? [Genre B]? What sounds right or wrong in the fusion?

**👂 The feeling** — What does it evoke? What does it do to you?

**✊ The stance** — Does it take a position on something? Too much? Not enough? Not readable?

Answer one question if you like — or all three. There's no right answer, only yours.

---

**How to leave feedback:**

1. Click the "open the Discussion" link above
2. Scroll to the comment box at the bottom of the page
3. Write what you think (one sentence is fine), then click **Comment**

**Prefer to leave a voice note?**

- **On desktop:** drag your audio file directly into the comment box — GitHub creates an audio player in the thread.
- **On iPhone:** open the link in the GitHub app → tap the comment box → attachment icon → select your voice memo.
- **On Android:** same thing with the GitHub app for Android.
- **No GitHub account:** send your voice note by message (WhatsApp, iMessage, Signal) — it'll be posted with your name.
