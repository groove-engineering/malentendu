# Fiche d'écoute — modèle

> Copier et adapter pour chaque morceau. Retirer les `[crochets]`.

---

🎵 **[Titre]** — [Genre A] × [Genre B]

**Écouter :** [lien SoundCloud]

**Laisser un retour :** [ouvrir la discussion sur GitHub]

---

Ce que je cherche chez vous :

**🎼 L'oreille** — Vous reconnaissez [Genre A] ? [Genre B] ? Qu'est-ce qui sonne juste ou faux dans la fusion ?

**👂 Le ressenti** — Qu'est-ce que ça évoque ? Qu'est-ce que ça vous fait ?

**✊ La position** — Est-ce que ça prend position sur quelque chose ? Trop ? Pas assez ? Pas lisible ?

Répondez à une question si vous voulez — ou aux trois. Il n'y a pas de bonne réponse, seulement la vôtre.

---

**Comment laisser un retour :**

1. Cliquez sur le lien « ouvrir la discussion » ci-dessus
2. Descendez jusqu'à la boîte de commentaire en bas de page
3. Écrivez ce que vous pensez (une phrase suffit), puis cliquez « Comment »

**Vous préférez laisser une note vocale ?**

- **Sur desktop :** glissez votre fichier audio directement dans la boîte de commentaire — ça crée un lecteur audio dans le fil.
- **Sur iPhone :** ouvrez le lien dans l'app GitHub → onglet « Conversation » → nouveau commentaire → icône pièce jointe → sélectionnez votre mémo vocal.
- **Sur Android :** même chose avec l'app GitHub pour Android.
- **Pas de compte GitHub :** envoyez-moi votre note vocale par message (WhatsApp, iMessage, Signal) — elle sera postée avec votre nom.
