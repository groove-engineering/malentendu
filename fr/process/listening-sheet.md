Ce que j'attends de vous :

**🎼 L'oreille** — Vous reconnaissez [Genre A] ? [Genre B] ? Qu'est-ce qui sonne juste ou faux dans la fusion ?

**👂 Le ressenti** — Ça évoque quoi ? Ça vous fait quoi ?

**✊ La posture** — Est-ce que ça prend position sur quelque chose ? Trop ? Pas assez ? Pas lisible ?

Répondez à une question si vous voulez — ou aux trois. Il n'y a pas de bonne réponse, juste la vôtre.

---

**Comment laisser un retour :**

1. Cliquez sur le lien « ouvrir la discussion » ci-dessus
2. Descendez jusqu'au champ de commentaire en bas de la page
3. Écrivez ce que vous pensez (une phrase suffit), puis cliquez sur « Comment »

**Vous préférez laisser une note vocale ?**

- **Sur ordinateur :** glissez votre fichier audio (.m4a ou .mp3) directement dans le champ de commentaire — ça crée un lecteur audio dans le fil.
- **Sur iPhone :** ouvrez le lien dans l'app GitHub → onglet « Conversation » → nouveau commentaire → icône pièce jointe → sélectionner votre mémo vocal.
- **Sur Android :** même chose avec l'app GitHub pour Android.
- **Pas de compte GitHub :** envoyez-moi votre note vocale par message (WhatsApp, iMessage, Signal) — je la posterai avec votre nom.
